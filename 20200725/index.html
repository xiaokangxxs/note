小康新鲜事儿

简单了解

  • 在 Windows 下，打开 “计算机”，我们看到的是一个个的驱动器盘符
  • 在 Linux 下，我们是看不到这些驱动器盘符的，我们看到的是文件夹（目录）

文件和目录

  • ls:列出目录的内容  cd:切换目录  mkdir:创建目录
  • touch:创建文件  rm:删除文件或目录  cp:复制文件或目录
  • mv:移动(重命名)文件或目录  cat:查看文件内容
  • more:分屏显示文件内容，每次只显示一页内容
  • less:与more类似
  • less可以按键盘上下方向键显示上下内容,more不能通过上下方向键控制显示
  • less不必读整个文件，加载速度会比more更快
  • less退出后shell不会留下刚显示的内容,而more退出后会在shell上留下刚显示的内容
[xiaokang@hadoop ~]$ ls
[xiaokang@hadoop ~]$ ls -l
[xiaokang@hadoop ~]$ ls -lha
[xiaokang@hadoop ~]$ cd ~
[xiaokang@hadoop ~]$ mkdir data
[xiaokang@hadoop ~]$ mkdir -p test/a
[xiaokang@hadoop ~]$ touch xiaokang.txt
[xiaokang@hadoop ~]$ rm -rf data/
[xiaokang@hadoop ~]$ cp xiaokang.txt xiaokang1.txt
[xiaokang@hadoop ~]$ mv xiaokang.txt xiaokang1.txt test/a/
[xiaokang@hadoop ~]$ cat example.txt
[xiaokang@hadoop ~]$ less example.txt
[xiaokang@hadoop ~]$ more example.txt

实用系列

  • date:查看系统时间  cal:查看日历  df:查看磁盘剩余空间
  • ps:查看进程  kill:停止进程  top:动态显示运行中的进程并且排序
  • ifconfig:查看计算机当前的网卡配置信息  ping:检测当前计算机到目标计算机之间的网络
  • tar:打包/解包  cut:切割文本  echo:在终端中显示参数指定的文字，通常会和重定向联合使用
  • |:管道  >:标准输出，会覆盖文件原有的内容  >>:追加，会将内容追加到已有文件的末尾
[xiaokang@hadoop ~]$ date +"%F %T"
[xiaokang@hadoop ~]$ cal -y 2020
[xiaokang@hadoop ~]$ df -h
[xiaokang@hadoop ~]$ ps -aux
[xiaokang@hadoop ~]$ kill 1124
[xiaokang@hadoop ~]$ top
[xiaokang@hadoop ~]$ ifconfig
[xiaokang@hadoop ~]$ ping www.baidu.com
[xiaokang@hadoop ~]$ tar -zxvf spark-2.4.5-bin-hadoop2.7.tgz -C /opt/software/
[xiaokang@hadoop ~]$ cat /proc/sys/kernel/random/uuid | cut -c 1-8
[xiaokang@hadoop ~]$ echo "小康&小康新鲜事儿"
[xiaokang@hadoop ~]$ echo "小康&小康新鲜事儿" >> example.txt
